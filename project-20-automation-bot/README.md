# Automation Bot

A Python project for automating simple file management tasks.

---

## Overview

This project demonstrates how to automate repetitive file tasks such as creating folders, moving files, and organizing content.

---

## Features

- Create folders automatically
- Move files into categorized folders
- Organize files by extension
- Handle missing files safely
- Display automation results

---

## Project Structure

- main.py → Main Python script
- files/ → Source files folder
- organized/ → Organized output folder
- README.md → Project documentation

---

## Technologies Used

- Python
- os
- shutil
