# Log Analyzer

A Python project for analyzing log files and extracting useful information.

---

## Overview

This project reads a log file and analyzes it to count errors, warnings, and other events using filtering techniques.

---

## Features

- Read log files
- Count ERROR, WARNING, INFO messages
- Filter specific keywords
- Display summary report
- Simple text-based analysis

---

## Project Structure

- main.py → Main script
- logs.txt → Sample log file
- requirements.txt → Dependencies
- README.md → Documentation

---

## Technologies Used

- Python
- File handling
- String processing
