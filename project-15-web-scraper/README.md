# Web Scraper

A Python project for extracting data from websites using BeautifulSoup.

---

## Overview

This project demonstrates how to scrape data from a website and extract useful information such as titles and links.

---

## Features

- Fetch web pages
- Parse HTML content
- Extract titles and links
- Display structured data
- Handle request errors

---

## Project Structure

- main.py → Main script
- requirements.txt → Dependencies
- README.md → Documentation

---

## Technologies Used

- Python
- requests
- BeautifulSoup
