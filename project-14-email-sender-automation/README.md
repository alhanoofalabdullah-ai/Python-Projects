# Email Sender Automation

A Python project for sending emails automatically using SMTP.

---

## Overview

This project demonstrates how to send emails using Python and SMTP for simple automation tasks.

---

## Features

- Send email using SMTP
- Use secure SSL connection
- Customize subject and message
- Automate email sending
- Handle sending errors

---

## Project Structure

- main.py → Main Python script
- requirements.txt → Project dependencies
- README.md → Project documentation

---

## Technologies Used

- Python
- smtplib
- ssl

---
