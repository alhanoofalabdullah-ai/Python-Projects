# 📦 Inventory Management System (PRO)

A professional CLI inventory system built using Python.

## 🚀 Features
- Add / Update / Delete products
- Search products
- Inventory reporting
- JSON data storage
- Logging system

## 🧠 Concepts Used
- Object-Oriented Programming (OOP)
- File Handling (JSON)
- Data Structures
- CLI Application Design

## ▶️ How to Run

cd project-08-inventory-system
python main.py
