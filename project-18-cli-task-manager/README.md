# CLI Task Manager

A professional Python command-line tool for managing tasks.

---

## Overview

This project demonstrates how to build a command-line task manager that allows users to create, view, update, delete, and complete tasks.

---

## Features

- Add new tasks
- View all tasks
- Update task title
- Mark tasks as completed
- Delete tasks
- Save tasks to JSON file

---

## Project Structure

- main.py → Main Python script
- tasks.json → Stores task data
- README.md → Project documentation

---

## Technologies Used

- Python
- JSON
- File handling
- CLI

---

```bash
python main.py
