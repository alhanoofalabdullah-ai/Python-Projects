# Data Cleaner

A Python project for cleaning and standardizing CSV data.

---

## Overview

This project demonstrates how to clean raw data by removing empty rows, trimming spaces, and standardizing text values in a CSV file.

---

## Features

- Read CSV files
- Remove empty rows
- Trim extra spaces
- Standardize text to lowercase
- Save cleaned data to a new CSV file

---

## Project Structure

- main.py → Main Python script
- raw_data.csv → Sample raw data
- cleaned_data.csv → Cleaned output file
- README.md → Project documentation

---

## Technologies Used

- Python
- CSV
- File handling
