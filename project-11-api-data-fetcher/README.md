# API Data Fetcher

A Python project that fetches data from a public API and displays it in a clean format.

---

## Overview

This project demonstrates how to connect to an external API, fetch JSON data, and display useful information using Python.

---

## Features

- Fetch data from public API
- Parse JSON response
- Display clean output
- Handle request errors

---

## Project Structure

- main.py → Main Python script
- requirements.txt → Project dependencies
- README.md → Project documentation

---

## Technologies Used

- Python
- Requests
- JSON
- APIs

## Setup Commands

- cd ~/Python-Projects
- mkdir project-11-api-data-fetcher
- cd project-11-api-data-fetcher
- touch README.md main.py requirements.txt

## Run the Project

- pip install -r requirements.txt
- python main.py
