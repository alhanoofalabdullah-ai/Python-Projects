# File Backup System

A Python project for creating backups of files and folders.

---

## Overview

This project demonstrates how to create a simple backup system using Python by copying files from a source folder to a backup folder.

---

## Features

- Copy files from source folder
- Create backup folder automatically
- Preserve file names
- Handle missing folder errors
- Display backup status

---

## Project Structure

- main.py → Main Python script
- requirements.txt → Project dependencies
- README.md → Project documentation

---

## Technologies Used

- Python
- shutil
- os
